Um projeto para um site de uma academia ficticia, chamada SilentGym.

O site utiliza html, css e javascript puro e permite criar e armazenar uma lista dos exercicios para fazer em determinada ficha.

Ele possui: 
- Armazenamento local: salvando no navegador o conteudo dos exercicios em cada uma das fichas.

- Criação dinâmica de exercicios: Permite ao usuário criar, editar e excluir, definindo o nome, a carga, e o número de repetições. Também adiciona uma checkbox para o usuário marcar os execicios já realizados no treino.

- Reponsividade: O site se ajusta para para diferentes tamnhos de tela (mobile, tablet, computador, etc..). Garantindo uma harmonia no layout, e uma melhor experiência pro usuário.
